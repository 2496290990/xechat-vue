# 1. 介绍 
> 当前项目架构 为Vue + ElementUI + Electron Node版本为14.17.0

# 2. 怎么启动
1. 安装依赖 
```shell
npm install
```
2. 运行 
```shell
npm run serve
```
3. 打包安装包
```shell
npm run electron:build
```
